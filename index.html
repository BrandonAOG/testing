Watch & devices

Two ways to connect a watch — no accounts, no subscriptions.

📡 Live heart-rate workouts (Bluetooth)

Works with chest straps and watches that broadcast heart rate (Garmin, Polar, Wahoo, many others — check your watch's "Broadcast HR" setting). Live BPM drives a real calorie burn estimate, then logs it as exercise. Needs Chrome or Edge; on iPhone use the free Bluefy browser.

⌚ Apple Watch / iPhone Health (via Shortcuts)

Apple doesn't let websites read Health data directly, but the built-in Shortcuts app can send it here. One-time setup (~2 min):

  1. Open Shortcuts → + → name it "Send Health to FitTrack"
  2. Add action Find Health Samples → type Steps, Today, Sum
  3. Add Find Health Samples again → Sleep (optional: another for Weight, latest)
  4. Add action Open URLs and build:

    replacing each value with the matching Health variable
  5. Run it any time — or add an Automation (e.g. daily 9 pm) so it syncs itself

When the link opens, today's steps, sleep, and weight drop straight into your log.
