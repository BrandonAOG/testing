Circuit Parameters
ⓘ Neutral Conductors — 310.15(E) A neutral that carries only the unbalanced load from other conductors in the same circuit is not counted as a current-carrying conductor for derating purposes. A neutral supplying predominantly non-linear loads (e.g. electric discharge lighting, harmonics-producing equipment) is counted per 310.15(E).
